# xiangmu1CodeFarm # 牧养记（Muyangji）

**中文名：牧养记**

「牧养记」是一款基于JavaFX开发的单机2D像素风格模拟经营游戏。你将以农场主的身份，在**五大交易市场**间奔波，低价购入幼崽，精心养殖，待牲畜成熟后选择**利润最高的市场**高价出售。

游戏核心特色在于**动态经济系统**：五个市场的牲畜价格独立随机波动，距离越远的市场潜在利润越大，但**路费成本**也越高。你需要精准判断价格走势，规划贸易路线，平衡饲料、棚舍、路费与收益，最终赚取最大利润，冲击经营评分排行榜！

## 核心玩法特色

### 🌍 五大动态市场
- **清风镇**（距离近，价格平稳）、**红叶城**（中等距离，波动适中）、**碧波港**（较远，水产偏好）、**金穗市**（远，谷物/牲畜高价）、**雪山驿**（极远，极端波动）
- 每个市场的牲畜**买入价**和**卖出价**每日独立随机浮动
- 同一头牲畜，在不同市场售价可能相差30%以上，套利空间巨大

### 🐄 牲畜养殖系统
- 多种牲畜（牛、羊、猪、鸡），完整生命周期（幼崽→成年→繁殖→衰老）
- 养殖成本真实模拟（饲料消耗、健康度、棚舍空间）

### 🛣️ 路费与距离机制
- 前往越远的市场，单次路费越高（如清风镇免费，雪山驿需20金）
- **赶集**时需支付往返路费，路费从交易利润中扣除或单独支付
- 策略抉择：是就近低价出售，还是冒险去远方博取高利润？

### 📊 经营评分与目标
- 综合总资产、跨市场交易次数、最高单笔利润等计算评分
- 动态目标引导玩家逐步解锁更远的市场

## 运行环境

- JDK 21 或更高版本
- 操作系统：Windows / macOS / Linux（JavaFX跨平台）
- IDE：IntelliJ IDEA（推荐）

---

## 快速开始

1. 克隆仓库并导入为Maven项目
2. 确保`pom.xml`中依赖正确下载
3. 运行`MuyangjiApplication.java`中的`main`方法
4. 首次启动自动创建数据库和初始数据（含五大市场及初始价格）
5. 开始你的牧场经营之旅！
