# xiangmu1CodeFarm 代码自动化农场游戏

基于编程算法驱动的自动化农场模拟游戏
通过编写代码控制农场种植、浇水、施肥、收割全流程，用算法优化收获效率，比拼程序思维与资源调度能力

项目简介

CodeFarm 是一款编程教育 + 模拟经营趣味小游戏。

区别于传统点击式农场游戏，本项目完全依靠玩家编写代码逻辑实现自动化农场运作：

- 玩家通过编写调度算法，控制土地播种、灌溉、养护、收割
- 系统模拟真实作物生长周期、资源消耗机制
- 根据 产量、资源利用率、时间效率 综合计算得分
- 玩家不断优化代码逻辑、迭代算法策略，实现收益最大化

项目旨在以游戏化方式锻炼循环、条件判断、资源调度、贪心策略、自动化流程设计等编程思维。

核心玩法

1.代码驱动农场
所有农场操作（种植、浇水、施肥、收割）均由玩家自定义脚本自动执行，无需手动点击。
2.真实作物生长模拟
每种作物拥有独立生长时长、缺水阈值、肥力消耗、产量属性，代码必须适配规则才能高效产出。
3.算法评分机制
系统会根据：
-单位时间总产量
-水资源、肥料利用率
-空地块闲置率
-无效操作冗余度
  综合结算分数，鼓励玩家优化代码、减少浪费。
  4.迭代优化玩法
  玩家可不断修改、调试、重构代码，对比每轮运行分数，逐步打磨最优自动化种植策略。

功能模块

-可视化农场地块系统（网格土地、作物状态实时展示）
-内置在线代码编辑器
-脚本沙盒安全运行机制
-作物生长模拟引擎
-自动化种植/养护/收割逻辑执行
-智能评分 & 算法效率评估
-运行日志、报错调试提示
-多套脚本保存与切换
-分数记录（可拓展排行榜）

技术栈

前端

- Vue3
- Canvas 2D（农场场景渲染、作物动画）
- 内置代码编辑器

后端

- Spring Boot
- MySQL（用户数据、脚本、分数记录）

核心技术亮点

- 脚本沙盒隔离运行：限制死循环、恶意代码，保证系统稳定
- 统一时间帧模拟引擎：保证作物生长与代码执行时序精准同步
- 算法量化评分体系：将代码优劣可视化、数据化

运行流程

1.进入游戏，打开代码编辑面板
2.编写自动化农场控制代码（播种、浇水、判断成熟、收割逻辑）
3.点击运行，引擎模拟农场时间流逝
4.脚本自动批量操作所有地块作物
5.结算本轮总产量与算法得分
6.根据分数优化代码策略，再次迭代冲击高分

项目特色

-  游戏化学编程：告别枯燥刷题，在经营游戏中练习算法思维
-  策略深度高：贪心算法、循环调度、资源分配均可影响最终分数
-  可视化反馈：每一次代码优化，都能直观看到产量提升
-  安全沙箱机制：避免错误代码导致卡死、崩溃
-  高拓展性：可后续新增天气系统、害虫系统、自动机器、地块解锁等玩法
